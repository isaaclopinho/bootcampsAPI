# DevCamper API

> Backend API for DevCamper application,
 which is a bootcamp directory website.

 ## Usage

Add a "Config.env" in config folder and update the value/settings to your own


## Install Dependencies

```
npm install
```

## Run app

```
#Run in dev mode
-> npm run dev

#Run in prod mode
->npm start

```
